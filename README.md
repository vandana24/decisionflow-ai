# DecisionFlow AI

Transform meetings into decisions, actions, and accountability.

## Overview

DecisionFlow AI is an enterprise AI agent that analyzes meeting transcripts and generates:

- Meeting summaries
- Key decisions
- Action items
- Risks
- Dependencies
- Follow-up emails

The goal is to help teams convert discussions into actionable outcomes.

## Features

- Upload or paste meeting transcripts
- AI-powered meeting analysis
- Decision extraction
- Risk identification
- Action item tracking
- Follow-up email generation

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Planned Enhancements

- Multi-agent architecture
- Microsoft Foundry integration
